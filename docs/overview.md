**AI + IoT 융합 솔루션 기획서 (4주 완성 전략)**

---

## ✍️ 핵심 피치

- **문제:** 장을 본 직후 영수증을 처리하지 못해 유통기한 관리 실패
- **해결:** 냉장고 옆 자석 부착 **ESP32-CAM 스캐너**에 영수증을 꽂으면, AI 모델이 자동 분류 및 유통기한 계산 → 앱 알림
- **차별점:** "폰으로 찍으세요"(X) → "냉장고 옆에 꽂으세요"(O). **3초 습관**을 공략하는 UX

---

## ⚙️ 기술 스택 (4주 집중 버전)

- **하드웨어:** ESP32-CAM + 링 LED + 3D 프린팅 케이스
- **백엔드:** Python Flask + Naver Clova OCR + Supabase
- **AI 모델:** `model.pkl` (Tfidf + Naive Bayes/SVM) + `expiry_logic.py` 규칙 엔진
- **프론트엔드:** React(Web) → Week 3 완성, Week 4에 React Native로 APK 포팅
